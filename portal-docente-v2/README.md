# Portal Docente CREO

Aplicación web React para la consulta y gestión de horarios académicos.

## Características
- Autenticación segura.
- Consulta de horarios en tiempo real desde Firebase.
- Panel administrativo con analítica básica.
- Diseño responsivo y moderno (Glassmorphism).

## Cómo subir a GitHub

1.  Crea un nuevo repositorio en GitHub (sin README, sin .gitignore).
2.  Ejecuta estos comandos en tu terminal:

```bash
git remote add origin https://github.com/TU_USUARIO/TU_REPOSITORIO.git
git branch -M main
git push -u origin main
```

## Desarrollo Local

```bash
npm install
npm run dev
```

## Construcción

```bash
npm run build
```
